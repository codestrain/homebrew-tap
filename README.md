# codestrain Homebrew tap

```bash
brew install codestrain/tap/codestrain
codestrain --all
```

Or pin the tap explicitly first:

```bash
brew tap codestrain/tap
brew install codestrain
```

Issue tracker: https://github.com/codestrain/codestrain-cli/issues
